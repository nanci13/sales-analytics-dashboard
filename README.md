# 📊 Sales Analytics Dashboard using Power BI

An interactive **Sales Analytics Dashboard** developed using **Microsoft Power BI** to transform raw sales data into meaningful business insights. This dashboard enables organizations to monitor sales performance, analyze revenue and profit trends, evaluate customer behavior, and support data-driven business decisions through interactive visualizations.

---

## 🚀 Features

- 📈 Sales Performance Analysis
- 💰 Revenue & Profit Analysis
- 👥 Customer Insights
- 🌍 Region-wise Sales Analysis
- 📦 Product Performance Analysis
- 📊 Monthly & Yearly Sales Trends
- 🏆 Top Selling Products
- 🎛 Interactive Filters (Slicers)
- 📌 KPI Cards
- 📉 Dynamic Charts and Visualizations

---

## 🛠️ Technologies Used

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset

## 📥 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/Sales-Analytics-Dashboard.git
```

### Open the Project

1. Install **Microsoft Power BI Desktop**.
2. Open the `Sales Analytics Dashboard.pbix` file.
3. Connect the dataset if prompted.
4. Refresh the data.
5. Explore the interactive dashboard.
